Hangboard Calculator and Equations

Sometimes these equations evaulate to something that doesn't really make sense, e.g. negative weights.
Sensible outputs mitigates that as follows:
  -no negative values; outputs 0 instead.
  -when calculating weight: if input edge is the same as the target edge, output the input weight.
  -when calculating edge: if the input weight is the same as the target weight, output the input edge.
  -when calculating edge: if the target weight is more than the maximum possible weight, output 0.
Whether you set sensible outputs or not, sometimes the equations will come up with no real values; in which case the output will just be 'X'.
